What can I do with this computer?

Now we all use computers every day, but sometimes it's difficult to work out what we can do with them. Wikipedia is a great resource, but it can be too difficult both in English and Czech. There is also the simple English Wikipedia which can be useful, though there are not as many pages to look up. If you are not sure what you want to read here, you could find an random article in Czech, English, or Simple English. Just like opening an encyclopedia at any page, you never know what you might find.

Fortunately, we have some content of our own that you can look at, and the Duhovka Elementary wiki is ready for you to both read and edit.

The first version of this web page is being written by Rob who runs the Geek Club, and so we would expect to see the programming language Scratch here. Other coding resources might be Code Academy where you can learn html & CSS which is a little more difficult than Scratch, and which is what Rob us using to write this page, and also git, which he is using to store it and to allow others to help him work on it (this is much more difficult). Maybe one day a version of this website which has been created by Rob and worked on by him and children in the school may be on the school website where anybody with a computer and an internet connection can see it. For now, though versions of it can be cloned or downloaded from git it is not in fact available directly on the internet. If you see it on your computer it is because it is saved on your computer.

If you can listen to sound on your computer right now, you could do worse than take a look at the video encyclopedia Check123 which has a number of 1, 2, and 3 minute videos on everything fromm the arts to science, history, and technology.
